# EML Intranet (KM) — Lean Skeleton

## Quick Start (local)
1) Install MkDocs + Material:
   ```bash
   pip install mkdocs mkdocs-material
   ```
2) Serve locally:
   ```bash
   mkdocs serve
   ```
3) Build static site:
   ```bash
   mkdocs build
   ```

## Structure
- `docs/` — content in Markdown
- `mkdocs.yml` — nav + theme config
